# 🏭 Goodwill Traders — Business Website

A complete business website developed to bring an established offline trading business online and attract modern buyers.

---

## 🚨 Problem

The business operated offline with **no digital presence**, making it invisible to younger and online-first customers.

This limited growth and reduced potential leads.

---

## 💡 Solution

* Built a **complete website from scratch**
* Added product/service listings for customer clarity
* Implemented a **lead capture system**
* Managed full deployment and hosting setup

---

## 📈 Outcome

* Created a **strong online presence**
* Improved visibility among modern buyers
* Generated **new customer inquiries through the website**

---

## ⚙️ Tech Stack

* React.js
* JavaScript (ES6+)
* Tailwind CSS
* EmailJS
* Netlify

---

## 🧠 Key Features

* Business-focused design
* Lead generation system
* Fully responsive layout
* Easy navigation for users
* End-to-end deployment handled
